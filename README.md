# Interact_Web3_Dapp
Practice ethereum web3 api interact  
Build a message board dapp on Ropsten Testnet   
[Demo](http://albertethdapp.nctu.me:9000/)

Step:
- git clone https://github.com/ksin751119/Interact_Web3_Dapp.git
- cd Interact_Web3_Dapp
- npm install
- bower install web3
